# Nasrat Nasrati Portfolio (Single Page Application)

This repository now contains a **single-page developer portfolio** built with React (CDN-based, no build step required).

## Run locally

1. Open `index.html` directly in your browser, or
2. Serve this folder with any static server.

Example:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Files

- `index.html` — app entry file
- `app.jsx` — React components and portfolio content
- `style.css` — styling and responsive layout
