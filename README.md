# Nasrat Nasrati Portfolio (Single Page Application)

This repository contains a **single-page React portfolio** (CDN-based, no build step).

## What's improved

- Hero section now shows your **GitHub profile image**.
- GitHub section now shows **live status** (repos, followers, following, account date) using GitHub API.
- Shows top repositories by stars.

## Run locally

```bash
python3 -m http.server 8080
```

Open: `http://localhost:8080`

## Files

- `index.html` — app entry file
- `app.jsx` — React components and live GitHub status logic
- `style.css` — styling and responsive layout
